Exposure times: what exposure times?

This is an experimental view of the exposure times of the Science observations made by Chandra. That is, how long does Chandra spend observing a target in a single observation, rather than a look at how much time is spent on each target. The data used is for all the data shown in the calendar view, and is split by proposal cycle, as well as there being one curve for all the data (the dotted line).
