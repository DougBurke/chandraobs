Here are some ways to explore the observations by Chandra:
  • by instrument,
  • by viewing the full SIMBAD object hierarchy or treating them individually,
  • by constellation,
  • and by proposal category.

You may need to hit return (or enter) twice; once to select the choice from the options, and then to actually run the search. There is also the slight problem that not all the suggested names will result in a match (due to “reasons”). These are being worked on!
